Help on Searching

Below are some sample searches intended to familiarize new users with the Archive's powerful information retrieval capabilities. Performing these searches will help you understand the differences between the various search options that the Archive has to offer.

Text Searching

  • Enter cloud worm and select or;

  • Enter eagle bowl and select and;

  • Enter "a thousand fighting men" [without quotation marks] and select phrase.

Image Searching

  • Find objects which contain any of your search terms:

    • Select eagle and bird.

  • Find only those objects which contain all of your search terms:

    • Select thel and willow.
